# dotFiles for my archGnome machine.

## Requirements.

Ensure to have the following installed on your system.

- Git. pacman -S git
- GNU Stow. pacman -S stow

## Installation.

1. Clone the dotfiles repo in you $HOME/dotfiles directory
2. Use GNU Stow to manage your symlinks.

```
> git clone https://github.com/KlesskGit/dotfiles.git
> cd dotfiles
> sotw .
```
